## 概要
タイトルを入力するだけでYouTube台本が自動生成されるWebアプリケーションを作成する

## 機能要件

### コア機能
1. **タイトル入力フォーム**
   - ユーザーがYouTube動画のタイトルを入力
   - 入力バリデーション（最大文字数など）

2. **AI台本生成**
   - Anthropic Claude APIを使用して台本を自動生成
   - 生成内容:
     - オープニング（視聴者の注意を引くフック）
     - 本編（構造化された内容）
     - クロージング（CTA付き）
   - 想定時間: 5-10分の動画

3. **台本表示・編集**
   - 生成された台本を見やすく表示
   - コピー機能
   - ダウンロード機能（テキストファイル）

### UI/UX
- シンプルで直感的なインターフェース
- レスポンシブデザイン（モバイル対応）
- ローディング状態の表示

### 技術スタック
- **フロントエンド**: React + TypeScript + Vite
- **スタイリング**: Tailwind CSS
- **AI**: Anthropic Claude API (Claude 3.5 Sonnet推奨)
- **デプロイ**: Firebase Hosting / Vercel
